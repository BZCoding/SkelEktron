Hello World!

This is SkelEktron, an unofficial Electron application template

Core Features

  • A smart cross-platform logger
  • A ready-to-go auto updater (you just need to hook in a working server)
  • Built-in support for Electron Builder
  • Built-in support for different build targets (ie. release, debug, test, etc) with different settings
  • Optional printer management, provided by printer

We are using node , Chromium , and Electron .
